# addLaTeX
クソボケが。いちいち日本語対応めんどくせえのよハゲが。死ね。

ってわけで色々まとめた個人用のVSCode向け拡張機能。
TeXファイル→PDFファイルにする時、日本語文字が含まれてたらソレ用にビルドレシピを切り替え。
ソレ以外にも諸々を自動化。
